Laboratorul 10

Tema 10.1

Se considera documentul html e1.html care contine un input de tip text cu id-ul "mesaj" si un buton "Salveaza mesaj" cu id-ul "salveaza". Scrieti cod JavaScript pentru a implementa urmatoarele:

  • La incarcarea paginii, dupa introducerea in input a unui mesaj personalizat si apasarea butonului "Salveaza mesaj" se va salva in localStorage mesajul din input, va aparea un alert cu mesajul "Mesajul a fost salvat!" iar continutul paragrafului cu clasa "afiseaza" se va actualiza cu noul mesaj.
  • Mesajul este afisat din localStorage in paragraful "afiseaza" ori de cate ori pagina este reincarcata.
  • La click pe body se va sterge mesajul din localStorage iar paragraful va reveni la continutul initial.

Tema 10.2

Se considera documentul html e2.html care permite utilizatorului sa introduca date personale (nume, email si varsta). Scrieti cod JavaScript pentru a implementa urmatoarele:

  • La incarcarea paginii, dupa introducerea datelor in campurile input si apasarea butonului "Salveaza detaliile" se vor salva informatiile ca obiect in localStorage, va aparea un alert cu mesajul "Datele au fost salvate!" iar informatiile salvate vor fi afisate sub forma de paragrafe (sau lista) in divul cu id-ul "informatii".
  • Informatiile din localStorage vor fi afisate in pagina ori de cate ori pagina este reincarcata.

Tema 10.3

Se considera documentul html e3.html si fisierele json bele.json, copii.json si psiho.json.Scrieti cod JavaScript pentru:
La incarcarea paginii, la selectarea unei optiuni din select se va face o cerere Ajax catre fisierul json corespunzator optiunii selectate. Datele din fisier se vor trimite in pagina si in sectiunea cu id-ul "carti" se vor crea imaginile cartilor selectate (elemente img) impreuna cu descrierea lor (elemente figcaption).

Tema 10.4

Se considera documentul html e4.html si fisierul json dictionar.json. Scrieti cod JavaScript astfel incat dupa introducerea in inputul de tip text a unui cuvant si selectarea unei optiuni, la click pe butonul de submit se va face o cerere Ajax catre fisierul json, datele din fisier se vor trimite in pagina si se va afisa traducerea cuvantului in limba selectata daca acesta se gaseste in dictionar si un mesaj in cazul in care nu exista cuvantul in dictionar.
