Laboratorul 6

Vom folosi fisierele reteta.html si reteta.css. Creati fisiere JavaScript pe care le legati de fisierul html, pentru a rezolva urmatoarele cerinte:

Tema 6.1

  • Creati un obiect JavaScript care reprezinta lista de cumparaturi pentru reteta de pateuri cu cascaval. Obiectul va contine ingrediente ca proprietati, iar valorile acestora vor reprezenta preturile ingredientelor.
  • Scrieti o functie care primeste ca parametru obiectul creat si returneaza valoarea totala a cumparaturilor. Afisati valoarea in consola.
  • Scrieti o functie care primeste ca parametru un ingredient si pretul acestuia si il adaugă in obiectul „lista de cumparaturi”. Afisati noul ingredient si pretul sau in consola.

Tema 6.2

  • Creati un array de obiecte numit retete, unde fiecare obiect reprezinta o reteta si contine urmatoarele proprietati: nume: numele rețetei (string), timpPreparare: durata preparării în minute (number), ingrediente: o listă de ingrediente principale (array de stringuri).
    Exemplu: 
      var retete = [
  { nume: "Pateuri cu cascaval", timpPreparare: 45, ingrediente: ["făină", "unt", "cașcaval", "ouă"] },
  { nume: "Salata de fructe", timpPreparare: 15, ingrediente: ["mere", "banane", "portocale", "struguri"] },
  { nume: "Supa de legume", timpPreparare: 30, ingrediente: ["ceapă", "morcovi", "cartofi", "pătrunjel"] },
  { nume: "Pizza", timpPreparare: 60, ingrediente: ["făină", "roșii", "brânză", "ciuperci"] },
  { nume: "Tort de ciocolata", timpPreparare: 90, ingrediente: ["făină", "ciocolată", "ouă", "zahăr"] }
];
  • De la prompt cereti utilizatorului sa introduca un nume de reteta. Afisati intr-un alert informatiile despre reteta în formatul: "Pateuri cu cașcaval" se prepară în 45 de minute. Ingrediente: făină, unt, cașcaval, ouă sau mesajul "Nu exista reteta" daca aceasta nu exista in vectorul de retete.
  • De la prompt cereți utilizatorului să introducă un timp maxim de preparare (in minute). Folosind metoda filter, creați un nou array care să contina doar retetele care pot fi preparate tn timpul specificat. Afitati rezultatele intr-un alert in formatul: "Retete care se pot prepara în maxim X minute: Pateuri cu cașcaval, Salată de fructe, Supă de legume"

Tema 6.3

  • Creati un array numit sfaturiDeGatit, care sa contina diverse sfaturi utile despre gatit sub forma de stringuri (minim 5). De exemplu: 
    
var sfaturiDeGatit = [
  "Întotdeauna citește rețeta complet înainte de a începe.",
  "Măsoară ingredientele cu precizie pentru cele mai bune rezultate.",
  "Gătește la foc mic pentru a evita arderea mâncării.",
  "Folosește ingrediente proaspete pentru un gust mai bun.",
  "Nu te teme să experimentezi cu condimentele!"
];
  • La fiecare încărcare a paginii, alege aleatoriu un sfat din array-ul sfaturiDeGatit și afișează-l ca text al elementului paragraf din headerul documentului, inlocuind textul existent.

Tema 6.4

La intrarea pe pagina se va afisa un prompt cu intrebarea, "Cum te numesti?" Dupa ce utilizatorul raspunde, title-ul tab-ului se va schimba in "Salut, nume!"

Tema 6.5

Scrieți o functie care selectează toate imaginile din sectiunea cu id-ul "galerie" si setează atributul lor src la "https://media.tenor.com/7qFULBHgzlYAAAAi/bubu-cooking-dudu-bubu.gif" iar atributul alt la "bubu gateste".

Tema 6.6

Scrieti o functie care schimba continutul elementelor figcaption in "Figura i", unde i este numarul de ordine al elementului figcaption.

Tema 6.7

Scrieți o functie care selectează toate elementele din lista cu pasii de preparare si scrie cu majuscule textul acestora.
