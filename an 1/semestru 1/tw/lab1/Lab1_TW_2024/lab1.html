Prezentare teoretică a limbajului HTML (Hypertext Markup Language)

  • descrie structura si continutul unei pagini web
  • contine elemente intre paranteze unghiulare = tag-uri ex: <a>, <img>, <p>, <table> (cuvinte cheie)
  • documentul are o structura ierarhică arborescentă
Exista doua tipuri de taguri:
  • taguri container : <numeTag> continut </numeTag> (ex: <p>Paragraf1</p>)
  • taguri goale: <numeTag> (ex: <img>)
  • in tagul de inceput pot aparea atribute = informatii suplimentare (ex: <img src="numeFisier.jpg" >)

Atribute:

  • sintaxa: <tag atribut1="valoare1" atribut2="valoare2">
  • pot fi globale = valabile pentru toate tagurile (ex: id, class, style)
  • sau specifice unor taguri (ex: src pentru tagul <img>, href pentru tagul <a>)

Structura fisier html:

<!DOCTYPE html>
<html>
<head>
 antetul paginii: informatii despre document
</head>

<body>
 corpul paginii: elementele vizibile din pagina
</body>
</html>

Taguri:

  • <!DOCTYPE> = versiunea html (<!DOCTYPE html> folosit pentru HTML5)
  • <html> = intreaga pagina din browser (<head> si <body> sunt tagurile copii)
  • <head>
    • contine informatii specifice despre document
    • conținutul nu este afisat in pagina web
    • tag-uri folosite in head: title, style, meta, link, script, etc.
  • <body> = continutul paginii
Taguri semantice: au efect de structurare al documentului (nevizibil)
  • <header> - descriere rapida de inceput a continutului paginii -ex: sigla, motto, denumire
  • <nav> - meniul principal al paginii
  • <section> - parti mari care au legatura intre ele
  • <article> - parti mari independente similare
  • <aside> - publicitate sau continut legat de text
  • <footer> - drept de copyright
  • <h1>Titlu cu font mare</h1> sau ...<h6>Titlu cu font mic</h6>
Taguri de grupare
  • <main> - defineste continutul specific al paginii (nu include elemente care se repeta pe pagina cum ar fi header, footer, nav)
  • <div> - defineste o sectiune/diviziune generala a paginii
  • <span> - defineste o sectiune/diviziune la nivelul unei linii
  • <p>Paragraf</p> - defineste un paragraf (poate contine doar text sau taguri "phrasing"- span, em, strong, a)
  • <figure> si <figcaption> - grupeaza o imagine/diagrama/ilustratie impreuna cu o descriere
Taguri pentru liste
<ul>- defineste o lista neordonata
<li>element1 lista neordonata </li>
<li>element2 lista neordonata </li>
</ul>

<ol>- defineste o lista ordonata
<li>element1 lista ordonata </li>
<li>element2 lista ordonata </li>
</ol>

atributul type - folosit pentru tipul markerului elementelor din lista ordonata
listele pot avea numai copii de tipul li, insa elementele li pot avea orice continut, inclusiv alte liste

Tagul pentru linkuri
<a href="calePaginaNoua">elementClickabilAncora</a> - defineste o hiperlegatura
  • href este cel mai important atribut: destinatia linkului (un URL sau un id al unui element din pagina)
  • alte atribute: target (cu valorile "_self/_blank/_parent/_top/nume-iframe"), download
  • poate contine alte elemente

Taguri media
<img src="caleFisier" alt="descriereImagine"> - defineste o imagine
  • atribute: src (numele sau URL-ul imaginii), alt (text alternativ care este afisat in caz de eroare)
  • alt este un atribut important deoarece ajuta utilizatorii care nu pot vizualiza imaginea
  • imi va permite sa adaug un titlu folosind atributele title
<audio> - include un fișier audio
<video> - include un fișier video
<embed> - include un fișier media

Tagul pentru tabele
<table> - defineste un tabel
<tr><td>dataRow1Col1</td><td>dataRow1Col2</td></tr>
<tr><td>dataRow2Col1</td><td>dataRow2Col2</td></tr>
</table>
(tr=tableRow si td=tableData)
  • in interior pot exista: <thead> (cap tabel), <tbody> (corpul tabelului), <tfoot> (subsolul tabelului)
  • atribute: border, cellspacing, cellpadding, width, height

Elemente multimedia
<iframe> - incarca o alta pagina in pagina curenta
  • atribute: src, width, height, frameborder
  • folosit in general pentru video-uri, slide-uri, etc.

Taguri meta
<meta name="description" content="...">
<meta name="keywords" content="...">
<meta name="author" content="...">
<meta name="viewport" content="width=device-width, initial-scale=1.0"> - pentru optimizarea pe dispozitive mobile

Tema 1.1

Creati o pagina reteta.html avand ca tema o reteta culinara folosind informatiile din textul de mai jos.

Indicatii privind crearea paginii:

  • folosind tagul <title> adaugati un titlu pentru pagina voastra
  • pentru structurarea documentului html folositi tagurile semantice (<header>, <section>, <article>, <aside>, <footer>, <h1>,...,<h6>) tinand seama de semnificatia acestora. Folositi si alte taguri sugestive pentru elementele din document (liste, tabel, paragrafe, ancore, etc.)
  • adaugati un favicon paginii
  • validati documentul html folosind validatorul html

2) Dupa ce ajunge la temperatura potrivita aluatul se taie in dreptunghiulete de dimensiuni egale.
3) Se rade cascavalul si se amesteca bine cu cele 2 oua. Se adauga in compozitie si sarea.
4) Compozitia se presara pe cate un dreptunghi de aluat. Deasupra se aseaza un alt dreptunghi de aluat

5) Pateurile se aseaza pe o tava cu hartie de copt. Se lasa la copt aproximativ 30 de minute.

Tema 1.2

Creati un fisier html care sa contina o prezentare detaliata a unui film preferat. Structurati documentul folosind tagurile semantice (<header>, <nav>, <section>, <article>, <aside>, <footer>).
Folositi si alte taguri sugestive pentru elementele din document (liste, tabel, paragrafe, imagini, video, iframe, ancore, etc.). Includeti informatii ca:

  • titlul filmului, o imagine reprezentativa (poster),
  • informatii generale despre film (gen, data lansare, durata, regizor, actori principali, premii),
  • o scurta prezentare (rezumat),
  • distributia,
  • galerie de imagini si video,
  • review-uri.

Includeti trailerul oficial folosind un element <iframe>.

Faceti legaturi la pagina filmului si la paginile actorilor de pe IMDb.

Idei de teme pentru proiect

  1. Rețete culinare:
    • Prezintă rețete pentru diferite feluri de mâncare, ingrediente, metode de preparare și fotografii.
  2. Sporturi:
    • Oferă informații despre sporturi diferite, echipe, statistici, antrenamente și știri recente.
  3. Călătorii:
    • Împărtășește destinații de călătorie, ghiduri, itinerarii, sfaturi de călătorie și fotografii din diferite locuri.
  4. Animale de companie:
    • Detaliază diferite rase de animale, îngrijire, hrănire și antrenament.
  5. Tehnologie:
    • Publică articole despre gadgeturi, recenzii de produse, sfaturi IT și noutăți din lumea tehnologiei.
  6. Cultură și artă:
    • Explorează opere de artă, biografii de artiști, recenzii de filme sau cărți, și evenimente culturale.
  7. Sănătate și fitness:
    • Oferă sfaturi despre dietă, exerciții fizice, stil de viață sănătos și recuperare.
  8. Grădinărit:
    • Prezintă tipuri de plante, tehnici de îngrijire a grădinilor, sfaturi pentru grădinărit urban.
  9. Modă:
    • Explorează tendințele în modă, stiluri vestimentare, sfaturi de îmbrăcăminte și accesorii.
  10. Hobby-uri și meșteșuguri:
    • Prezintă diverse hobby-uri, cum ar fi tricotatul, pictura, sculptura, etc., împreună cu tutoriale.
  11. Evenimente locale:
    • Oferă informații despre festivaluri, concerte, expoziții și alte evenimente comunitare.
  12. Știință și educație:
    • Publică articole despre descoperiri științifice, experimente, educație și resurse de învățare.
